# word2vec_frxxz

本项目旨在使用gensim库训练一个word2vec模型，使用其副产品词向量。

数据集使用小说《凡人修仙传》。

word2vec_train.py用以训练模型。

word2vec_predict.py用以使用模型。

word2vec_draw.py用以做一个展示，展示word2vec词向量表示词汇上的有效性，其结果如图。
